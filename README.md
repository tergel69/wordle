# Wordle Clone

A Wordle-inspired word guessing game built with React and Next.js.

## Features

- Multiple word lengths (3-9 letters)
- Color-coded feedback system
- Statistics tracking per mode
- Share functionality
- Responsive design

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)
